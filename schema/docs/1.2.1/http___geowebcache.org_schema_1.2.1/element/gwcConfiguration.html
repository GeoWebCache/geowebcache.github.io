element gwcConfiguration

Documentation
    Wrapper element for XStream. Make sure it has the correct namespace
Properties
This component is not nillable.
Model
<gwcConfiguration>
(version?, backendTimeout?, cacheBypassAllowed?, runtimeStats?, httpUsername?, httpPassword?, formatModifiers?, gridSets, layers )
</gwcConfiguration>

Nested Element Summary
xs:integerbackendTimeout
           The backend timeout is the number of seconds GWC will wait for a backend server to return something before closing the connection.  
xs:booleancacheBypassAllowed
          Determines whether cached=false is allowed for requests going through the WMS service, including converters such as Google Maps. 
gwc:formatModifiersformatModifiers
          These are the global format modifiers that apply to all layers in this file, unless the layer has separately defined modifiers. 
 gridSets
           The list of grid sets provided by this configuration.  
xs:stringhttpPassword
          If you wish to have every connection to HTTP backends use HTTP Authentication set this to the username. 
xs:stringhttpUsername
          If you wish to have every connection to HTTP backends use HTTP Authentication set this to the username. 
 layers
           The list of WMS layers provided by this configuration.  
xs:booleanruntimeStats
          Runtime statistics run, by default, every three second and provide data about how many requests the system has been serving in the past 3, 15 and 60 seconds, as well as aggregate numbers. 
xs:stringversion
           The version number should match the XSD namespace and the version of GWC
