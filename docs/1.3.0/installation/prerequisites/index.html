Installing Java and TomcatÂ¶

This section will describe how to install a Java Runtime Enivronment (JRE) and Apache Tomcat (a Java Servlet Container) on the most common platforms. You can skip this step and go to Installing GeoWebCache if you already have a servlet container running. While this section only discusses the Tomcat servlet container, the setup is usually very similar for other servlet containers.

Warning

Many Linux distributions come with packages for Java and Tomcat. However, these are often an incorrect version and are frequently configured to deny filesystem access, hence it is recommended to install these packages manually.
